Fideya Arsip adalah sistem manajemen arsip berbasis web yang dirancang untuk membantu pengelolaan data dan dokumen secara terstruktur, cepat, dan aman. Sistem ini mempermudah proses penyimpanan, pencarian, dan pelacakan riwayat dokumen sehingga pekerjaan administrasi menjadi lebih efisien.

  - Dokter / User: hanya dapat melihat arsip tanpa dapat mengubah data
- Menampilkan detail riwayat berdasarkan NRM (Nomor Rekam Medis) atau ID arsip lain
- Tampilan antarmuka sederhana dan mudah digunakan
